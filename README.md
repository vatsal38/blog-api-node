APIs of the blogs with token user autantication.

## Getting Started

First, add the node modules:

```bash
npm i
```

Run the development server:

```bash
npm run start:dev
```

## APIs

- Create user API.
- Login user API.
- Get user by Id.
- List all user.
- Create Blog.
- List of all blogs.
- Get blogs By user Id.
- Get blog by blog ID and user Id.


## Technologies Used

- Node.js 
- Express.js
- MondbDB

## .env file

```bash
MONGODB_URI="mongo connection url"
NODE_ENV="development"
JWT_SECRET="your key"
PORT=5000
```
